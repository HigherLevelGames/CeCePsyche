nTools: A free collection of simple utilities, 
by Nikkolai Davenport <me@nikkolai.com>

This package adds a Tools menu to the menu bar in Unity. From this Tools menu, the following utilities can be selected:

1. Create Basic Folders
Creates a basic folder structure for new projects consisting of 22 essential directories.

2. Edit Prefs Utility
Opens a utility for editing key-value pairs in PlayerPrefs and EditorPrefs.

3. Delete PlayerPrefs
Deletes all player preferences.

4. Delete EditorPrefs
Deletes all editor preferences (use with caution).
